# 🔮 Customer Churn Prediction System

A complete Machine Learning project to predict customer churn using 
Logistic Regression and Random Forest with ~90% accuracy.

![Python](https://img.shields.io/badge/Python-3.9+-blue)
![Flask](https://img.shields.io/badge/Flask-2.3-green)
![scikit-learn](https://img.shields.io/badge/scikit--learn-1.3-orange)

## 📌 Features

- ✅ **Two ML Models**: Logistic Regression + Random Forest
- ✅ **~90% Accuracy** on test data
- ✅ **EDA & Visualizations**: Correlation heatmaps, distribution plots
- ✅ **Feature Engineering**: Risk scores, derived features
- ✅ **REST API**: Single & batch predictions
- ✅ **Web Interface**: Beautiful UI to test predictions

## 📊 Model Performance

| Model | Accuracy | AUC-ROC |
|-------|----------|---------|
| Logistic Regression | ~88% | ~0.93 |
| Random Forest | ~91% | ~0.96 |
